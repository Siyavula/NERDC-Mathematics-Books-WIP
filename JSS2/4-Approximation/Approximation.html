Rounding off: an introduction

Given the number here, round off to 11 decimal place: 5,5855,585.

Answer:
numeric
STEP: <no title>
[−1 point ⇒ 0 / 1 points left]

When a number has lots of decimal digits, we sometimes round off to simplify the number. Rounding off changes the number, but only by a small amount.

The first thing is to figure out which of the digits in the number tells us what to do: we always look at the digit one place further along than we are rounding the number to. In this question we are rounding off to 11 decimal place so we must look at the value of the second decimal to decide what to do.

It is helpful to separate the number into two parts by drawing a line (|) between the first decimal and the second decimal, like this:

5,585⟶5,5|85

If the number following the line is less than 5, we should round down; if it is greater than or equal to 5, we round up.

In this case, the number after the line is "8." The 8 tells us to round up, which means up from 5,585 to 5,600 (the number gets larger - that is why it is called rounding "up.")

We write this as: 5,585≈5,6. The symbol "≈" means "approximately equal to."

The correct answer is: 5,6.

Answer:
numeric
STEP: <no title>
[−1 point ⇒ 0 / 1 points left]

We sometimes round off numbers to simplify them. Rounding off usually changes the number, but only by a small amount.

The first thing is to figure out which of the digits in the number tells us what to do: we always look at the digit one place further along than we are rounding the number to. In this question we are rounding off to the thousandths place - which is the third decimal place - so we must look at the value of the fourth decimal to decide what to do.

If the number following the line is less than 5, we should round down; if it is greater than or equal to 5, we round up.
