In words, this says: "The set of rational numbers is the set of all number which can be written as fractions aa over bb such that aa and bb are elements of the integers and bb is not equal to zero."

Note that rational numbers may not always look like fractions! The definition says that the numbers can be written as a fraction if you want it to. In fact, the rational numbers include lots of numbers which may not look like fractions. Here is a summary of what this means:

  • the set of rational numbers includes:
    1. all fractions with integers in the numerator and denominator, as long as the denominator is not zero, for example: 12;−711;2005012;−711;20050
    2. all integers because integers can be written as fractions, for example 5=515=51
    3. all terminating decimal numbers, for example: 4,2;−1,5;3,509274654,2;−1,5;3,50927465
    4. all decimal numbers with a recurring pattern, for example: 0,3333…;2,6˙;6,5827827827…0,3333…;2,6˙;6,5827827827…
  • the set of rational numbers does not include:
    1. any fraction where the denominator is zero
    2. decimal numbers which go on forever without a repeating pattern, for example: 2,4681012…;3,14159265…2,4681012…;3,14159265…

The correct choices for this question are ℚQ and D (from the table).
