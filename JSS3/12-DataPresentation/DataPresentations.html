Remember the “pie” represents some kind of whole, which is made up of the slices. Add up all the slices and we get the complete pie. Here the slices known as sectors are divided into values represented as fractions. All the fractions have the same denominator; thus, to get whole "pie", we need to end up with a value like this: 23232323. This evaluates to 1, the whole pie. So xx will be 423423.

To find the number of degrees a sector should be in a pie chart you can follow these steps:

  • Find the total amount used in the data in this case it was 23
  • What ratio of this total is represented by this sector? Here it was 423423
  • Multiply your ratio with 360 °° (Remember a full revolution = 360 °°)
Sector°=423×360°=62,61°Sector°=423×360°=62,61°

So xx has a fraction value of 423423, which is the same as 62,61°°.
